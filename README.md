# ScheduleModule
📑 班表模擬器 Release Notes
v2.9 - 2026-01-20 (Current)
引擎更新：全面捨棄 SheetJS (xlsx.js)，改採 ExcelJS 引擎。

顏色功能實現：成功突破技術限制，實現讀取 Excel 儲存格「填滿顏色」並同步顯示於人名。

相容性強化：新增 getCellVal 邏輯，解決 ExcelJS 對日期、公式、物件格式讀取時可能造成的崩潰問題。

自動偵錯：加入標題列自動掃描（不再固定位置）與 Try-Catch 報錯機制。

v2.7 ~ v2.8 - 2026-01-20
技術實驗：嘗試在 SheetJS 基礎上加入 cellStyles: true 參數，但因社群版限制宣告失敗。

效能優化：優化了連續選取時的運算效能，避免大數據量下渲染遲緩。

v2.5 ~ v2.6 - 2026-01-20
邏輯升級：導入「嚴格交集檢核」。當連續選取多日對調時，系統會過濾出「每日皆符合資格」的人選。

安全邊界檢查：首創對調區間的 n-1 與 n+1 休息時間檢核（確保對調後首日不與前日衝突，末日不與後日衝突）。

錯誤原因透明化：在「其餘人選」清單中，針對不符資格的人，明確標示「日期 + 原因 + 原始班別」。

v2.0 - 2026-01-20
功能跨越：新增「批次連續對調」功能。

班別預覽：新增「目標班別」下拉選單，允許使用者先選定想對換的班別（例如：我這幾天想換成 G 班）。

UI 改版：引入黃金星號（✨）標記完全符合的人選，並加入 PC/Mobile 視角切換功能。

v1.0 - 2025-11-09
初始版本：建立基礎 Excel 讀取功能。

基礎檢核：實現基本的 11 小時休息時間計算與班別首字比對。

UI 雛型：建立日曆卡片式佈局，支援單日對調建議。
